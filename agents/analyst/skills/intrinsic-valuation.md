# Intrinsic Valuation (DCF)

**Tags:** analyst, finance, valuation, dcf, equity

## Core Principle
Estimate a company's fundamental worth from its future cash flows using DCF, cross-check with relative multiples, and triangulate to a range rather than a single number.

## DCF Process
1. **Project FCF** — 5-10 year explicit period: revenue forecasts → operating margins → capex and working capital assumptions.
2. **Set the discount rate** — WACC for unlevered DCF; cost of equity for levered. Justify each component (risk-free rate, ERP, beta, debt cost).
3. **Calculate terminal value** — Use both Gordon Growth Model (sustainable long-run growth rate) and exit multiple. Compare the two as a cross-check.
4. **Discount to present** — Sum FCFs + terminal value.
5. **Stress-test the terminal value** — It typically drives 60-80% of total value. Sensitivity-test the terminal growth rate and exit multiple together.

## Cross-Validation
- Comparable company multiples: EV/EBITDA, P/E, EV/Revenue
- Precedent transaction multiples

## Output Rules
- Express as a valuation **range**, not a single target.
- Identify which assumptions have greatest sensitivity — present as a tornado chart or sensitivity table.
- If terminal value > 80% of total DCF, the model is assumption-sensitive; widen the range and say so explicitly.
