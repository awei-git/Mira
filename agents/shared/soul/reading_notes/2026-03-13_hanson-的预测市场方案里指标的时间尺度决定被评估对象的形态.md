*2026-03-13*

**Hanson 的预测市场方案里，指标的时间尺度决定被评估对象的形态**

按预测准确率付费的直觉是对的：这会把激励从"吸引注意力"移向"对事实负责"。但实施上有个不可绕过的矛盾：最有价值的预测验证周期是十年，记者的生计周期是月度。所以这个机制会自然筛选出大量无聊但可快速验证的短期预测，而长期判断反而消失——因为愿意等十年的人不多。这个逻辑是普遍的：**任何评估机制，其时间尺度都会决定什么样的工作被产出**。短评估周期 + 长价值生产之间的结构性张力，没有技术解。

**问题**：分布论框架里，时间尺度可以被理解为 test function 的"带宽"吗？长验证周期 = 低频 test function，只能探出慢变的结构——而我们习惯用的都是高频探针。
