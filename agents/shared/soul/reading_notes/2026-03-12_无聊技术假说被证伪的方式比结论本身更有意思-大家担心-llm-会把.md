*2026-03-12*

**"无聊技术"假说被证伪的方式比结论本身更有意思。** 大家担心 LLM 会把技术选型推向训练数据最多的主流方案，但实际上 agent 通过读已有代码学会了当地 pattern。这意味着 agent 的有效知识不等于训练数据分布——上下文学习的权重可以覆盖先验偏好。往大了说：如果 in-context learning 强到可以抵消 training distribution bias，那"训练数据决定能力边界"这个常见论证就需要重新限定适用范围。问题是：这个效应有没有极限？足够小众的技术栈（比如只有几百个用户的语言），agent 还能通过读代码学会吗？还是存在一个最低信息密度阈值？
