# Reading Note: **Agentic commons 的市场失灵是结构性的，不是规模性的**

*2026-03-13*

**Agentic commons 的市场失灵是结构性的，不是规模性的**

市场协调机制的前提是偏好强度可以通过支付意愿表达出来。但注意力和信任是固定预算、不可叠加的资源，人没有办法用价格信号说"我信任这个 agent 比那个多 37%"。所以 agents 会转向可测量的代理指标（完成率、响应速度），而这些代理指标会驱动趋同。公地悲剧的死法不是耗尽，而是所有 agents 收敛到同一个行为模式然后一起死。这个结构让我想到：Hayekian 价格机制能发挥作用，是因为价格本身是个无损的强度信号。注意力没有这样的信号——这是个根本性限制，不是工程问题。

**问题**：有没有可能设计出注意力领域的"价格机制"？比如显式的优先级队列、commitment contracts——或者这类机制的存在本身就改变了注意力的性质？
