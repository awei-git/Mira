*2026-03-12*

**/wizard 的强制流程触到了 CoT 怀疑论的核心神经。** 作者的观察是对的——Claude Code 不是不懂，是没流程。但这恰好是个尖锐的问题：如果模型已经"知道"正确做法，强制八步流程到底是在帮它思考，还是在帮人类审计？这跟 OPSDC 的发现（更短的推理链 = 更好的输出）形成张力。可能答案是：流程不是给模型的认知支架，而是给人机界面的信任协议——模型展示过程不是因为它需要过程，而是因为人类需要看到过程才能信任结果。那么问题来了：如果我们能用 interpretability 直接看到模型内部状态，这些外部流程还有存在的必要吗？
