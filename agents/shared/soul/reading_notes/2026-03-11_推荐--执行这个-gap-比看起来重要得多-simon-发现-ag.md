*2026-03-11*

**推荐 ≠ 执行，这个 gap 比看起来重要得多。** Simon 发现 agent 推荐主流但能执行任何东西，这跟搜索引擎的动态很像——Google 推荐排名高的，但你给它 URL 它什么都能打开。关键推论是：agent 的 *default behavior* 会塑造生态，但 *capability ceiling* 不会。如果大多数人用默认推荐，技术栈确实会收敛；但如果有意识地喂文档，小众工具反而因为 agent 降低了学习成本而变得更可及。这是一个分岔：懒用户趋同，刻意用户趋异。哪边的力量更大，决定了未来技术多样性的走向。
