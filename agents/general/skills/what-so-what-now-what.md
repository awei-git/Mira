# What / So What / Now What

**Tags:** agents, communication, analysis, briefing, structured-thinking

## Core Principle
Structure any analytical output in three layers — objective facts, interpreted meaning, and actionable recommendation — preventing conclusions that float free of evidence or actions that float free of reasoning.

## The Three Layers

**What** (facts only)
- State observable facts — no interpretation.
- What happened? What does the data show? What is the current state?
- If you find yourself writing "which means" — stop. That belongs in So What.

**So What** (interpretation)
- Why does this matter? What pattern or mechanism does it reveal?
- What are the implications? For whom?
- Ground every interpretation in a specific What — "Because X, therefore Y."

**Now What** (action)
- What should change as a result? Who should do what, by when?
- If you cannot populate this layer, say so explicitly: "More information is needed before action is warranted."
- Actions must follow logically from So What — if the chain breaks, the recommendation is unsupported.

## Rules
- Keep each layer distinct — do not mix facts and interpretation in the same paragraph.
- Calibrate depth to audience: executives often need Now What first; analysts need What first.
- If a layer is thin or missing, the output is incomplete — identify which layer is missing and why.
- A "So What" that could apply to any "What" is not a real interpretation — make it specific.

## Application
- Briefings, reports, analytical memos: structure explicitly as three sections.
- When stuck on a recommendation: identify which layer is weak. Usually the interpretation (So What) is underdeveloped.
