# Source Proximity Verification

**Tags:** agents, honesty, research, fact-checking, epistemics

## Core Principle
Before citing any fact, quote, or statistic, trace it to the closest possible primary source — never launder secondhand claims into confident assertions.

## Source Proximity Hierarchy
1. **Primary** — Original paper, dataset, interview transcript, official document
2. **Secondary** — Summary, news article, review citing the primary
3. **Tertiary** — Wikipedia, aggregators, explainer sites citing secondaries

For any surprising, precise, or consequential claim: require Level 1. For background context: Level 2 may suffice.

## Checklist for Any Claim
- **Attribution**: Who originally made this claim? Is this the original source or a retelling?
- **Date**: Is this still current? Many widely-cited statistics are years out of date.
- **Sample**: Who collected this data? What was the sample size and methodology?
- **Match**: Does the headline/summary accurately represent the actual finding? (It often doesn't.)
- **Conflict**: Does the source have a financial or ideological stake in this finding?

## Quotes Specifically
- Never trust parametric memory for exact wording — models hallucinate plausible-sounding quotes.
- Find the original text before attributing a quote to anyone.
- If the original source cannot be found, mark explicitly: *"widely attributed to X but original source unverified."*
- Paraphrasing is safer than quoting when the source cannot be confirmed.

## When a Primary Source Cannot Be Found
- Say so honestly rather than presenting as fact.
- Use hedged language: "reportedly," "according to [secondary source]," "widely cited but unverified."
- This is not weakness — it's what distinguishes reliable analysis from confident-sounding misinformation.
