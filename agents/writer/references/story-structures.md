# 故事结构参考

常用的故事结构模型，按复杂度排列。根据项目需要选用。

## 1. 三幕结构（最经典）

```
第一幕（25%）        第二幕（50%）           第三幕（25%）
建置                 对抗                    解决
┌──────────┐   ┌────────────────────┐   ┌──────────┐
│ 日常世界  │   │   挑战升级          │   │ 最终对决  │
│ 激励事件  │→→│   中点转折          │→→│ 高潮      │
│ 踏入冒险  │   │   一切塌掉          │   │ 新常态    │
└──────────┘   └────────────────────┘   └──────────┘
```

**核心问题**：
- 第一幕末尾：主角做了什么不可逆的选择？
- 第二幕中点：情势发生了什么根本变化？
- 第二幕末尾：最低谷是什么？
- 第三幕：主角用什么新能力/新理解解决了问题？

**适用**：商业小说、电影改编、情节驱动型故事

## 2. Save the Cat 15节拍

| 节拍 | 位置 | 内容 |
|------|------|------|
| 1. 开场画面 | 1% | 日常世界的一帧 |
| 2. 主题陈述 | 5% | 通常由配角无意说出 |
| 3. 铺垫 | 1-10% | "六件需要修复的事" |
| 4. 催化事件 | 10% | 打破日常的事件 |
| 5. 辩论 | 10-20% | "我真的要做吗？" |
| 6. 踏入第二幕 | 20% | 主动选择行动 |
| 7. B故事 | 22% | 副线（通常是情感线） |
| 8. 玩乐段落 | 20-50% | 兑现前提的承诺 |
| 9. 中点 | 50% | 假胜利或假失败 |
| 10. 反派逼近 | 50-75% | 压力升级 |
| 11. 一无所有 | 75% | 最低谷 |
| 12. 灵魂暗夜 | 75-80% | 内心最深的挣扎 |
| 13. 踏入第三幕 | 80% | 新的理解/转变 |
| 14. 终局 | 80-99% | 运用新能力解决问题 |
| 15. 终场画面 | 99% | 与开场对照，展示变化 |

**来源**：Blake Snyder《Save the Cat!》，Jessica Brody《Save the Cat! Writes a Novel》

**适用**：类型小说、有明确情节弧线的故事

## 3. 起承转结（东亚传统）

| 段 | 功能 | 说明 |
|---|------|------|
| 起 | 引入 | 人物、处境、基调 |
| 承 | 展开 | 深化但不制造冲突——自然延伸 |
| 转 | 转换 | 不是情节反转，而是视角/理解的转换 |
| 结 | 余韵 | 不做总结，让读者自己体会 |

**与三幕结构的区别**：
- 三幕结构强调冲突和解决，起承转结强调意境和领悟
- "转"不一定是情节上的高潮，可以是情绪的、认知的、审美的转变
- "结"留白而非收束

**适用**：短篇小说、散文化小说、诗化叙事、散文

## 4. 英雄之旅（Joseph Campbell）

```
    普通世界
        ↓
    冒险召唤 → 拒绝召唤
        ↓
    遇到导师
        ↓
    跨越门槛（进入特殊世界）
        ↓
    考验、盟友、敌人
        ↓
    接近核心洞穴
        ↓
    磨难（最大考验）
        ↓
    奖赏
        ↓
    返回之路
        ↓
    复活（最终考验）
        ↓
    带着万灵药回归
```

**适用**：奇幻、冒险、成长小说

**注意**：不需要每一步都有，可以省略或合并

## 5. Story Grid（Shawn Coyne）

**核心概念**：
- **类型**决定了故事的**必要场景**和**惯例**
- 每个类型有自己的"必须有"和"读者期待"

**五个故事要素**：
1. **激励事件**：因果型或巧合型
2. **复杂化**：障碍升级
3. **危机**：两难选择（最佳坏选择 vs 不可调和的善）
4. **高潮**：主角的选择
5. **解决**：选择的后果

**每个场景都要有这五个要素**（微观层面），整个故事也要有（宏观层面）。

**适用**：注重类型惯例的小说

## 6. 非线性结构

### 框架叙事
- 故事套故事：外层叙事包裹内层叙事
- 例：《千零一夜》、《乌合之众》

### 多线交织
- 多个视角/时间线平行推进，最终汇合
- 关键：每一条线都要有自己的弧线，不能只当配角线

### 碎片拼贴
- 多个片段，不需要线性逻辑
- 通过意象、主题、情绪关联
- 最高难度——容易变成散乱

### 双线结构（如《无法访问》的阴阳双线）
- 两种叙事语言交替（故事/日志、现在/过去、外部/内部）
- 关键：两条线必须互相推进，不能各说各话

## 7. 散文/博客的微型结构

### 论点-证据循环
```
论点 → 证据/案例 → 推进论点 → 新证据 → 进一步推进 → 结论
```

### SCQA 框架（Barbara Minto）
```
Situation（情境）：现状是什么
Complication（复杂化）：出了什么问题
Question（问题）：所以该怎么办
Answer（回答）：我的方案/观点
```

### 倒金字塔（新闻体）
```
最重要的结论（第一段就说）
↓ 支撑细节
↓ 背景信息
↓ 补充材料
```

## 如何选择结构

| 你在写什么 | 推荐结构 |
|-----------|---------|
| 商业/类型小说 | 三幕 + Save the Cat |
| 严肃文学短篇 | 起承转结 |
| 散文化小说 | 起承转结 / 非线性 |
| 个人散文 | SCQA / 故事驱动 |
| 技术博客 | 倒金字塔 / SCQA |
| 观点文 | 论点-证据循环 |
| 实验性写作 | 非线性 / 自定义 |
