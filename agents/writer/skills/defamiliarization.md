# 陌生化（Defamiliarization）

## One-liner
通过改变观察视角，把熟悉的事物变陌生 -- SF独有的认知翻转能力。

## The Problem
人类对熟悉事物会产生"感知自动化" -- 你不再真正看到你每天看到的东西。日常生活、社会制度、人类行为，我们对这些太习惯了，以至于不再质疑。SF的独特力量是：它可以让你重新看到这些东西，因为它可以改变观察者的参考系。

## The Pattern

```
1. SELECT   — 选择一个你想让读者重新看到的"熟悉事物"
               （某个社会习俗、人类行为、日常物品、制度...）
2. SHIFT    — 改变观察视角。常见的视角转换：
               - 外星人视角（从未见过这种事物）
               - 未来人视角（这种事物已经消失了）
               - AI/机器视角（用不同的逻辑理解这种事物）
               - 平行世界视角（这种事物从未存在过）
3. DESCRIBE — 用新视角的逻辑和语言描述这个事物。
               不是"假装不认识"，而是"用另一套认知框架理解"。
4. REVEAL   — 通过这种陌生的描述，揭示我们平时忽略的真相。
```

## Examples

**Example 1: 陌生化"工作"**
正常描述："他每天去上班。"
陌生化（外星人视角）：
"这些生物每个清醒周期都自愿前往指定地点，执行重复性任务长达8个时间单位，换取一种抽象符号。这些符号本身不能食用或提供庇护，但整个种族似乎相信它们具有价值。最令人困惑的是，大多数个体表示他们不喜欢这个过程，但仍然继续。他们称之为'生活'。"
（揭示：工作制度的荒谬性。读者重新看到了自己每天在做的事。）

**Example 2: 陌生化"葬礼"**
正常描述："我们去参加了葬礼。"
陌生化（AI视角）：
"他们把一个不再运行的有机体放进一个精心装饰的容器里，然后聚集起来围着它执行一系列声音输出 -- 有些是语言，有些似乎是功能失调的呼吸模式。整个过程的目的似乎不是为了那个已终止的个体（它显然无法接收任何输入），而是为了让其他个体的情感参数达到某种我无法建模的状态。"
（揭示：仪式的非理性本质，但也暗示了人类情感的深度 -- AI无法建模的东西。）

**Example 3: 陌生化"语言"**
正常描述："他们说着不同的语言。"
陌生化（未来人视角）：
"古人把思想编码成嘴部肌肉的连续运动，产生气压波动，另一个人的耳膜接收后再解码。这个过程的带宽极低 -- 大约每秒39比特 -- 且无法并行，所以他们经常在等待对方编码完成时产生焦虑。他们称这些为'对话'，并认为这是关系的基础。"

## Techniques

**1. 功能描述法**
不用名称，只描述事物的功能。
不说"椅子"，说"一个支撑有机体后肢弯曲角度的结构"。
注意：只在有叙事目的时使用。为陌生化而陌生化会变成噱头。

**2. 比例偏移**
改变某个维度的比例，让熟悉事物的荒谬性显现。
正常："我们工作到退休。"
偏移："这个种族把70%的清醒存在时间用于为他人的目标工作，剩余30%中的大部分用于恢复体力以继续工作。"

**3. 错误类比**
用观察者自己世界的概念来错误类比我们的事物。这种"错误"本身揭示了两个世界的差异。
外星人把"学校"类比为"幼体思想标准化设施"。这是"错误的"，但也是一种诚实。

**4. 选择性盲点**
观察者能看到我们忽略的东西，但对我们认为重要的东西毫无反应。
AI注意到人类呼吸节奏和情感状态的精确相关性（我们不注意），但完全忽略"礼貌"的概念（对AI没有功能意义）。

## When to Use
- 想让读者质疑某个"理所当然"的现实时
- 写外星人、AI、未来人的POV时（这些角色天然适合陌生化）
- 社会批评/讽刺类SF（陌生化是讽刺的最好工具之一）
- 开篇hook：用陌生化描述打开故事，让读者立刻进入SF的认知模式

## Common Pitfalls
- **为陌生化而陌生化**: 如果没有叙事目的，陌生化就是做作。每次使用都应该揭示某个被忽略的真相。
- **太聪明了**: 如果读者花太多时间解码你的陌生化描述，叙事节奏就断了。要在"有趣"和"可读"之间找平衡。
- **单一视角疲劳**: 整本书都是外星人看地球会让读者累。陌生化是调味料，不是主食。穿插使用。
- **讽刺过度**: 陌生化天然带有讽刺性。如果你的故事需要的是同理心而不是讽刺，注意控制tone。有时候陌生化的目的不是嘲笑人类，而是让人重新感动。
