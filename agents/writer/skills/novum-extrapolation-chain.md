# Novum推演链

## One-liner
从一个核心设定变化（novum）出发，系统性推导二阶、三阶、四阶后果 -- 科幻构思的核心思维工具。

## The Problem
大多数平庸SF只推到一阶后果就停了。"有了永生技术" → "人们不再害怕死亡"。然后整个故事都建立在这个浅层推演上。但真实世界中，任何重大变化都会产生一连串意想不到的后果。一阶后果是显而易见的；二阶以后才是故事的富矿。

## The Pattern

```
1. DEFINE NOVUM  — 明确你的核心设定变化。越精确越好。
                    不是"未来科技"，而是"成本为零的瞬间传送"。
2. FIRST ORDER   — 列出最直接的后果（3-5个）。
                    这些是大多数人会想到的。
3. SECOND ORDER  — 对每个一阶后果，问："然后呢？"
                    这里开始出现意外。
4. THIRD ORDER+  — 继续推。到第三层通常会出现
                    反直觉的、有趣的、适合写故事的后果。
5. CHECK         — 检查后果之间的一致性。
                    如果A和B互相矛盾，你需要解决冲突。
6. SELECT        — 从推演树中选择最有故事潜力的分支。
                    不是所有后果都需要写进小说。
```

## Example: 推演"免费瞬间传送"

```
NOVUM: 任何人可以瞬间传送到任何地方，零成本。

一阶:
  - 交通运输业消亡
  - 地理边界失去意义
  - 任何人可以瞬间到达任何地点

二阶:
  - 港口城市、机场城市经济崩溃
  - 国家主权概念被动摇（无法控制边境）
  - 犯罪变得极难预防（罪犯瞬间逃离）
  - 房地产逻辑彻底重写（"位置"不再重要）

三阶:
  - 新的权力结构：谁控制传送技术谁就是新政府
  - 隐私成为最稀缺的资源（任何人可以出现在任何地方）
  - 监狱制度失效 → 刑罚制度必须重新发明
  - 战争形态改变：不再有"前线"，任何地方都是战场
  - 文化同质化加速 → 反过来催生极端地方主义

四阶:
  - "住所"概念消失，人变成纯粹的游牧者
  - 亲密关系的含义改变（"我选择在你身边"不再意味着牺牲）
  - 自然保护区需要全新的保护机制
```

**第三、四阶出现的后果（隐私危机、亲密关系重新定义、极端地方主义）比一阶（交通消亡）有趣得多。** 好故事通常住在这里。

## Techniques

**1. "然后呢？"连锁法**
对每个后果持续追问"然后呢？"直到你到达一个让你自己惊讶的地方。如果你还没惊讶，你还没推够深。

**2. 交叉后果**
两个看似无关的一阶后果在二阶或三阶交叉时，往往产生最好的故事张力。
例：传送 → 犯罪难预防 + 传送 → 隐私消失 → 全民监控社会的合理化。

**3. 反向推演**
从一个你想写的场景/结局反推：要使这个场景合理，世界需要什么规则？这些规则又预设了什么novum？

**4. 输家视角**
每个技术变革都有赢家和输家。刻意从输家的角度推演后果。
传送技术的赢家：旅行者、逃犯。输家：航空从业者、边境守卫、隐士。
输家的故事通常比赢家的更有戏剧张力。

**5. Heinlein的"If This Goes On"**
取一个当前真实趋势，把它推到逻辑极端。
当前趋势：算法推荐 → 推到极端：人的所有决策都由算法做出 → 再推：自由意志成为奢侈品/犯罪行为。

## When to Use
- 构思阶段（在写之前把世界想清楚）
- 发现故事感觉"浅"的时候 -- 你可能停在了一阶
- 需要给世界增加复杂度但不知道从哪里加的时候
- 检查设定一致性（推演链中的矛盾 = 你的世界有bug）

## Common Pitfalls
- **推演不等于全写出来**: 你推到四阶是为了理解世界，不是为了全部塞进小说。用Incluing选择性地展示。
- **推演成了空想**: 每个推演步骤都应该能回答"这对一个具体的人意味着什么？"如果回答不了，那只是抽象的世界设定，不是故事素材。
- **忽视人性不变量**: 技术变了但人的基本需求（爱、安全、意义）没变。好的推演最终要回到人身上。
