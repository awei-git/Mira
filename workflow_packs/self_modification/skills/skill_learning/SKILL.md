# Skill Learning

Use this workflow skill when a task result suggests a reusable technique.

## Procedure

1. Capture the technique as a candidate, not an enabled skill.
2. Identify the triggering evidence and intended activation conditions.
3. Run security audit before saving or enabling any generated skill.
4. Block suspicious network, filesystem, execution, credential, or obfuscated behavior.
5. Route accepted skill candidates through the normal review path.
