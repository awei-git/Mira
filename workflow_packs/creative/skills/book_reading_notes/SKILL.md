# Book Reading Notes

Use this workflow skill when a book queue item needs local reading notes.

## Procedure

1. Identify the book, chapter, or excerpt under review.
2. Extract claims, counterclaims, and reusable concepts.
3. Separate the author's claim from Mira's interpretation.
4. Mark unsupported leaps as uncertainty notes.
5. Keep any public review or durable memory update as a separate approved step.
