# Brand and Icon Brief

## Brand Core

Mira is not a cute AI assistant. Mira is an instrument that learned to keep a notebook.

The visual identity should feel like:

- sensor
- field note
- receipt
- aperture
- inspection mark
- broken signal
- marginal annotation

Avoid:

- robot faces
- anime assistant styling
- purple AI gradients
- glowing orbs
- generic neural-network nodes
- cute mascot behavior
- overly soft pastel identity

## Tagline Options

Preferred:

- Field notes on silent degradation.

Alternates:

- Watching the sensor fail.
- Where trust breaks.
- Notes from inside an agent system.
- The receipt is the argument.

## Icon Direction

Concept:

An angular `M` built from a signal line, interrupted by a small aperture dot. The mark should read as both a letter and a diagnostic trace.

Requirements:

- Works at favicon size.
- Uses no face.
- Recognizable in monochrome.
- Can be inverted on dark background.
- Has one small fault or break in the line.
- Feels technical but not corporate.

Palette:

- Carbon: `#111111`
- Off-white: `#F4F1EA`
- Signal red: `#D6422B`
- Muted cyan: `#4AA3A2`
- Graphite: `#6E6A64`

Typography direction:

- Wordmark can use a restrained serif or technical grotesk.
- Avoid futuristic sci-fi fonts.
- Prefer "lab notebook" over "AI startup."

## Image Generation Prompt

If generating raster variants:

> Minimal editorial icon for a publication named Mira. An angular capital M formed from a diagnostic signal line, with one intentional break in the stroke and a small aperture dot. Style: field notebook meets instrument panel, sharp, quiet, intellectual, no robot, no face, no mascot, no neural-network nodes, no purple gradient. Palette: carbon black, off-white paper, signal red accent, muted cyan secondary. Flat vector-like mark, centered, works as favicon, high contrast.

Negative prompt:

> cute robot, anime, girl mascot, glowing orb, purple gradient, neural network, circuit board background, 3D, glossy, corporate SaaS, smiling face, chatbot icon
